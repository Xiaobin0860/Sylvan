Sylvan Documentation

Sylvan is a set of cross-platform tools for working with Xiangqi engines. It consists of:

  • sylvan—a graphical user interface.
  • sylvan-cli—a command-line interface for automating Xiangqi engine matches.
  • libsylvan—a library providing an interface for working with Xiangqi engines.

Sylvan is developed using C++ and Qt. The project has been in development since Dec 2019 and is currently in active development.

Status

sylvan

The graphical user interface currently only supports very basic features. Most of the work is currently done in sylvan-cli and libsylvan projects. We plan to focus on the GUI after the library has stabilised.

sylvan-cli

We consider sylvan-cli to be stable and usable for automating Xiangqi engine matches.

libsylvan

The library is under heavy development and the API is still changing.
